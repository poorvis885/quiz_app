🧠 Smart Quiz Platform

A modern, interactive web-based quiz application built using HTML, CSS, and JavaScript.
This platform allows users to take quizzes across multiple categories, track performance, and view progress analytics.

🚀 Live Demo

👉 https://poorvis885.github.io/quiz_app/

📌 Features
🎯 Multiple Categories
Technology
General Knowledge
Maths
English (Verbal Ability)
SQL / DBMS
Programming Languages
AI / ML
🧪 70+ Questions
10 questions per category
⏱️ Timer-Based Quiz
Each question has a time limit
📊 Progress Tracking
Scores saved using LocalStorage
Category-wise performance tracking
📈 Dashboard Analytics
Attempts per category
Average score calculation
🎨 Interactive UI
Smooth animations
Dark/Light theme toggle
🛠️ Tech Stack
HTML
CSS
JavaScript (Vanilla JS)
LocalStorage (for data persistence)
